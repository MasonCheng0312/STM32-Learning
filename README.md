## 硬體環境 (Hardware)
- **MCU**: STM32F407VGT6 (ARM Cortex-M4)
- **Board**: STM32F407G-DISC1 (Discovery Board)
- **Debugger**: ST-LINK/V2 (On-board)
- **Peripherals**:
  - USB to TTL (CP2102)
